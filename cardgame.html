No 1 About Project

Lord of War is a strategy based game, where players click on their randomly disbursed characters to fight opposing players in turn based order.

This project was our first venture into programming and creating webpages. This webapage was created simply using HTML and jQuery. Jquery is a dom based, client-side manipulation methedology used to make dynamic pages with simple imputs and minimal client and server side interaction. All the logic is proccessed in the clientside app and is temporarily stored on the client interface.

No 2 The Process

Process to designing the game was fairly straightforward. But the technical application of data interaction through jQuery was the toughest challenge/

Since all of the data is ephemeral, I needed to come up with a solution of how to create manipulations through jQuery. What I came up with was a two way binding of the css visual divs which create the cards and the sample library of character parameters. When you click on a card and click on an opponent card, the jQuery is sourcing the two fight opponents based on the identification marker on the div and then compairing it to the database id marker. The two characters "fight" based on the attacking turn.

CSS was used to style the cards. Each card is a PNG graphic overlayed on a second layer PNG graphic and then sized over a div with a color background. All these elements are moved together when selecting a card. Html is used for the structure and javascript and jQuery is used for the move and attack logic.
